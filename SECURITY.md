# Security Policy

## Supported Versions

Currently, only the `main` branch (latest release) is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Knowledge Sphere, please DO NOT open a public issue. 

Instead, please send an e-mail to the maintainer directly or use the GitHub Security Advisory "Report a Vulnerability" feature. We treat security issues with the highest priority and will attempt to address and release a patch as quickly as possible.

### What to include in your report:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Any suggested mitigations.
