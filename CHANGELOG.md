# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-11

### Added
- Complete end-to-end AI Document Intelligence pipeline.
- Google Gemini 2.0 Flash integration for rapid embedding and content generation.
- Dynamic fallback mechanism to OpenAI and Ollama for API resilience.
- Neo4j Knowledge Graph entity extraction and interactive React visualization.
- ChromaDB `v5` collections with 3072-dimensional vector search.
- JWT-based Authentication and user management.
- Robust Celery distributed task queue for asynchronous PDF processing.
- One-click Docker deployment configuration.
- Comprehensive API documentation via FastAPI.
- End-to-end production validation script.
